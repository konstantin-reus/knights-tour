# Project: Knight-tour

## Type
new-project

## Description
Develop a website, which visualize solving https://en.wikipedia.org/wiki/Knight%27s_tour problem.
It should be a web page working on Google Chrome browser. The webpage must display an empty chess board. Then, one knight is added on a random cell (never start with the same cell twice).
After that, the knight starts traversing all the cells available, never visiting the same cell twice (see https://en.wikipedia.org/wiki/Knight%27s_tour for conditions).
The next cell to jump must be chosen by Warnsdorf's rule. The process of traversing must be visualized on the board during the execution. 
Next to the board must be available counters, showing number of jumps since the starting position, number of starting positions already checked, number of solutions found, and the current start position.
There must be a slider to control speed of visualization, so a human being can observe it.
The visited cells must be marked on the chessboard.
